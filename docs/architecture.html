3 · AI Processing — Anthropic API ✓ Enterprise contract · No training data usage · No retention
Research Step
/api/leads/[id]/generate (step 1)
PII sent to Anthropic:
company name · signal type · visited URLs
account notes · crEnrichment · sfContext
existing research (if any)
Tool: web_search · 60s timeout · 2000 tokens
Cached 14 days on account
← writes: webResearch, industry, headcount, companyType
Draft Generation
/api/leads/[id]/generate (step 2)
PII sent to Anthropic:
name · title · company · LinkedIn URL
research summary · crEnrichment · sfContext
visitedUrls · account notes
Last 5 refinement examples (before/after — contain PII)
No tools · 30s timeout · 1500 tokens
+ Apollo product context · writing rules
← writes: emailSubject, emailDraft, linkedinNote
Refinement
/api/leads/[id]/refine
PII sent to Anthropic:
current draft text (contains name, company)
rep's feedback instruction
No tools · 30s timeout · 800 tokens
← writes: emailDraft or linkedinNote
← also saves to RefinementExample table
Account Research (manual refresh)
/api/accounts/[id]/research
PII sent to Anthropic:
company name · account notes · sourcedVia
Two calls: company research + job signals
No tools · 30s timeout · 1000 tokens each
← writes: webResearch, jobSignals, metadata
Insights / Patterns
/api/insights

→ returns: conversational brief (not stored)
Rule Suggestion
/api/settings/suggest-rule
PII sent to Anthropic:

current rules list
No tools · 30s timeout · 300 tokens
→ returns: suggested rule (not auto-saved)

Auth & Access Control
Browser Session
Single shared password (SITE_SECRET)
Cookie: v1_session · 1 year · httpOnly · secure · sameSite=strict
Middleware validates on every request
Fails closed if SITE_SECRET not set
Ingest Webhooks
Common Room: x-commonroom-webhook-secret header
Omni Sheet: x-ingest-secret header
Generic: Authorization: Bearer header
All compared to INGEST_SECRET env var
Bypasses session auth (no cookie needed)
Env Var Secrets
ANTHROPIC_API_KEY — Anthropic auth
INGEST_SECRET — webhook auth
SITE_SECRET — login password + session value
WEBHOOK_URL — Google Sheet endpoint
DATABASE_URL + TURSO_AUTH_TOKEN — Turso
All stored in Vercel encrypted env vars
